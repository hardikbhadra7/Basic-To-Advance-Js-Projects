# Kanban Board (HTML, CSS, JavaScript)

This is a simple Kanban Board project built using **Vanilla JavaScript**.

## Features

- Add new tasks
- Delete tasks
- Drag and Drop tasks between columns
- Task counter for each column
- Data saved using LocalStorage

## Columns

- To Do
- In Progress
- Done

## Technologies Used

- HTML
- CSS
- JavaScript (DOM Manipulation)
- Drag and Drop API
- LocalStorage

## Purpose

This project was created for learning JavaScript concepts like:

- DOM manipulation
- Event handling
- Dynamic element creation
- Drag and drop functionality
- Local storage
